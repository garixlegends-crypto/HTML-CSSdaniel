Heure suivante

Ecrivez un programme qui demande une heure à un utilisateur sous la forme de trois informations (heures, minutes, secondes).

il affiche ensuite l'heure qu'il sera une seconde plus tard.

Les erreurs de saisie doivent être gérées.

    Ce programme est moins simple qu'il en a l'air C-O-U-R-A-G-E : >
  • 14h17m59s => 14h18m0s
  • 6h59m59s => 7h0m0s
  • 23h59m59s => 0h0m0s (minuit)

Résultat :

var heure = parseInt(prompt('une heure')); var minute = parseInt(prompt('une minute')); var secoude = parseInt(prompt('une seconde')); if(isNaN(heure)){ alert('stop'); } if(heure == 23 && minute == 59 && secoude+1 > 59){ heure = 0; minute = 0; secoude = 0; } if(secoude+1 > 60 && minute == 60 || secoude+1 > 59 || heure > 24){ alert('stop'); heure = ""; minute = ""; secoude = ""; } else if(secoude+1 > 59 && minute == 59 ){ heure += 1; minute = 0; secoude = 0; } else if(secoude+1 > 59){ minute += 1; secoude = 0; } else{ secoude += 1; } console.log(heure,minute,secoude)
