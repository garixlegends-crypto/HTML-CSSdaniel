Nombre magique :

  1. message de bienvenue sur la console
  2. générer un nombre aléatoire entre 1 et 100
  3. faire apparaitre une boite de saisie
  4. Entrée de l'utilisateur d'un essai
  5. Si le nombre est trouvé, le jeu s'arrête
    Sinon le programme indique si le nombre magique est plus petit ou plus grand que la saisie
    Si l'entrée n'est pas un nombre, un message d'erreur s'affiche
  6. L'utilisateur à 6 essais, puis le jeu s'arrête et donne la solution.
console.log('bienvenue'); var nombre = Math.floor(Math.random() * 100) + 1; var reponse = 0; for(i=0; i<6; i++) { reponse = Number(prompt("Entrez un nombre entre 1 et 100")); if(reponse > nombre){ alert('plus petit'); } if (reponse < nombre) { alert('plus grand'); } if (reponse === nombre) { alert('bien jouer'); } if(isNaN(reponse)){ alert('pas de lettre') } } if (reponse !== nombre ) { alert( nombre); }
