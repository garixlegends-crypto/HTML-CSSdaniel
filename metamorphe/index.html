Meta-donees

la balise meta permet de donne des métadonnée dans un site web qui et pas visuellement visible sur le site internet avec beaucoup d'outil ou autre qui nous donne des informations sur notre site web

Balise Description
meta http-equiv="refresh" la balise meta refresh permet d'actualiser
la page de façon automatique apres quelque seconde
meta name="description"
Les administracteur peuvent definir leur mots de cles par une recherche
meta name="verify-v1"
il vérifie si l'utilisateur est bien le proprietaire du site
meta name="noimageindex"
La page n'apparait pas en réference
meta name="follow"
Le robot peut voir les liens dans la page
